# Restaurant Data Analysis

This project performs Exploratory Data Analysis (EDA) on a restaurant dataset to discover patterns in cuisines, ratings, votes, pricing, and services offered by restaurants.

## Objective
The goal of this project is to analyze restaurant data and extract useful insights that help understand customer preferences, restaurant popularity, and service availability.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset Information
The dataset contains information about restaurants including:

- Restaurant Name
- City
- Cuisines
- Price Range
- Aggregate Rating
- Votes
- Online Delivery
- Table Booking
- Geographic Location (Latitude and Longitude)

## Analysis Performed

### Level 1 – Basic Analysis
- Top cuisines analysis
- City-wise restaurant distribution
- Price range distribution
- Online delivery analysis

### Level 2 – Intermediate Analysis
- Restaurant ratings distribution
- Cuisine combination analysis
- Geographic analysis

### Level 3 – Advanced Analysis
- Highest and lowest voted restaurants
- Votes vs rating relationship
- Price range vs restaurant services

## Key Insights
- Identified the most popular cuisines served by restaurants.
- Analyzed cities with the highest number of restaurants.
- Observed the distribution of restaurant ratings.
- Explored the relationship between votes and ratings.
- Studied how price range influences online delivery and table booking services.
