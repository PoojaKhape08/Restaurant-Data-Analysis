# Level 3 - Advanced Data Analysis

This level focuses on advanced insights and relationships between different variables in the dataset.

## Tasks

Task 1: Restaurant with Highest and Lowest Votes  
Identify restaurants with the highest and lowest number of votes.

Task 2: Votes vs Rating Relationship  
Analyze the correlation between restaurant ratings and number of votes.

Task 3: Price Range vs Services Analysis  
Analyze how price range affects online delivery and table booking services.
