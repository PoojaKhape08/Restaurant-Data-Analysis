# Level 1 - Basic Data Analysis

This level focuses on basic exploratory data analysis of the restaurant dataset.

## Tasks

Task 1: Top Cuisines Analysis  
Identify the most common cuisines served by restaurants.

Task 2: City Analysis  
Analyze the distribution of restaurants across different cities.

Task 3: Price Range Distribution  
Understand how restaurants are distributed across different price ranges.

Task 4: Online Delivery Analysis  
Analyze how many restaurants provide online delivery and compare their ratings.
